# Daily Scraped Data

This folder contains the daily scraped data obtained from the Kijiji website. The data is collected using the [Extraction Program](../extraction_program) and saved in CSV format.

## File Naming Convention

The files in this folder follow the naming convention `kijiji_data_<YYYY-MM-DD>.csv`, representing the date of extraction.

## Data Format

The CSV files contain the following columns:

- Vehicle Make
- Vehicle Model
- Vehicle Year
- Vehicle List Price
- Vehicle Color
- Vehicle Condition (used/new)
- Vehicle Body Type (hatchback/SUV/crossover/etc…)
- Vehicle Wheel Configuration (4x4/AWD/RWD/etc…)
- Vehicle Transmission
- Vehicle Fuel Type
- Vehicle Mileage
- Vehicle VIN Number
- Vehicle Image Link (the first image [the hero shot])
- Seller’s Address

## Usage

- Use this data for analysis, research, or any other purposes related to vehicle information.
